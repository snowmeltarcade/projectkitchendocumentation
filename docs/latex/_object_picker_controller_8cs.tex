\begin{DoxyCompactList}\small\item\em Manages the object picker. This works by having an invisible plane collider that a ray from the mouse position intersects. An empty game object is placed at the hit location. Any object that the player selects will have its transform set as a child of that game object. This helps to ensure that the height of the selected object is above any actual objects in the scene. \end{DoxyCompactList}\end{DoxyCompactItemize}
