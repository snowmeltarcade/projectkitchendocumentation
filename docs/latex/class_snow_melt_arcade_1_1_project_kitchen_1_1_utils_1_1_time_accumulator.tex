\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_snow_melt_arcade_1_1_project_kitchen_1_1_utils_1_1_time_accumulator}{Time\+Accumulator}} stores spans of time over time. Unlike System.\+Time\+Span, it stores precision beyond milliseconds. Time is accumulated in milliseconds. 

Internally, this stores milliseconds as a float and not double. Double gives us too much precision which makes comparing against the total milliseconds a bit more challenging.
