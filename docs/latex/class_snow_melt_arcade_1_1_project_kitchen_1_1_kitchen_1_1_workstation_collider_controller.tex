\doxysubsection{Detailed Description}
Creates the workstation collider. 

If each workstation has its own box collider, these colliders touch each other when the workstations are next to each other. When a sphere collider, like food uses, rolls over the sides of two workstations, it will bounce or have an unintuitive collision. We are trying to simulate a continuous flat surface, but this is not what is happening. We need to create an actual single flat surface over all workstations to avoid this problem.
